
# Perfect Crop

A website to predict the crop to be grown

Link to website -> https://perfect-crop.onrender.com/




## Build

To deploy this project run

```bash
  python3 -m pip install -r requirements.txt
```

## Deploy

To deploy this project run

```bash
  gunicorn app:app
```
